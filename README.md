## 简单好用的小程序版Pokedex

### pokedex有哪些功能？

* 简单好用的宝可梦图鉴
* 收录了从第一世代到第八世代的全部1016只宝可梦
* 支持按中文名、编号搜索宝可梦
* 支持按属性、世代、蛋组筛选宝可梦
* 清晰直观的种族值、进化信息、弱点与抗性查询
* 新增675种招式展示，并提供搜索筛选功能
